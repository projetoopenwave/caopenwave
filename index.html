Instruções de Instalação (Cliente)

Para usuários finais, basta instalar o Certificado Root (acima).

1. Baixe o arquivo .crt desejado (Interno ou Global).

2. Clique duas vezes no arquivo e selecione "Instalar Certificado".

3. Selecione "Máquina Local" (Requer Admin) para aplicar a todo o sistema.

4. Escolha "Colocar todos os certificados no repositório a seguir".

5. Selecione "Autoridades de Certificação Raiz Confiáveis" e conclua.
